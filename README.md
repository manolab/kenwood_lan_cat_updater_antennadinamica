## cat_updater_lan890.py

### Run (using `pipenv`)

```shell
pipenv run python main.py --host <host> --port <port> --output <output_path> --user <username> --password <password>

## E.g.
pipenv run python main.py --host "192.168.1.16" \
  --port 60000 \
  --output "/usr/local/share/ad/frequency.txt" \
  --user "admin" \
  --password "normando"
```
